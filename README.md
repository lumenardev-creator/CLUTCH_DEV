# Clutch

Clutch is a modern landing page project built with HTML, TSX, and Tailwind CSS. It is designed to provide users with a seamless and premium experience.

## Sections

The landing page features smooth-scrolling navigation with the following key sections:

- **Home**: The landing hero area.
- **How Clutch Works**: An overview of the product mechanics.
- **Why Clutch**: Benefits and value proposition.
- **Clutch Score**: Information about scoring or reputation metrics.
- **Get Started**: Call-to-action to begin using Clutch.

## Tech Stack

- **Core**: HTML5, TSX (React/Next.js components)
- **Styling**: Tailwind CSS for rapid UI development and modern design aesthetics.

## Project Structure

- `clutch_landing_page*.html`: The main HTML landing page mockups and integration.
- `components/`: Contains reusable TSX components (e.g., `shared/Footer.tsx`).
- `hero/`: Hero section-specific assets or code.
- `logo/`: Product logo assets.

## Getting Started

1. Clone or download the repository.
2. Open the `clutch_landing_page*.html` file in your browser to view the static site, or run your standard local development server (like `npm run dev`) if you have a Next.js/React environment set up for the `.tsx` components.

## UI/UX Guidelines

- The design follows a modern "pill-shaped" UI design.
- All animations respect `preferred-reduced-motion` for accessibility.
- CSS variables are typically used for easy theme switching.
- A consistent 8px grid system is maintained throughout the design layout.
- Semantic HTML5 elements are prioritized.
